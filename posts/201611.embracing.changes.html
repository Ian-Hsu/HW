拥抱网络，拥抱变化

Ian，《新经济，新规则》读后

最近（2017年8月13日）听朋友说了一个真实的创业成功案例：某太阳能电路板的供应商做了七八年没什么起色，直到去年成为共享单车的零件供应，仅一年时间为合伙人带来不菲收入。无独有偶，另外一位朋友讲述了外卖包装盒公司最近两年在外卖平台销售增势下也营收暴增。有几个启示：1，坚持打造核心产品；2，做未来的市场；3，做能联网的设备；4，做标准零部件。其中2、3和4会在本文反复被申明。这是一篇2016年的读后，现在稍作整理、回顾，分享。

一些商业模式，从电话、传真机、打印机发明的年代开始，已经存在相当长的时间。

硬件 + 耗材的模式，几十年前的打印机和墨盒的搭配，因近十年的苹果手机 + APP商店、Kindle + 电子书、智能电视 + 视频内容，甚至在线游戏 + 道具的模式常常被提出来类比。

联网产品的模式，以电话与传真机为例，随着接入电信网的设备数量的增长其作用越得到体现，与如今人们加入社交网络、在线游戏一样：加入某网络的玩家越多，个体和平台本身越有价值。

互联网产品的免费模式，变现方法除了卖广告，还有收费增值服务。

读凯文·凯利（Kevin Kelly）的文集《新经济新规则》，可以获得更多在网络时代，关于技术、市场策略与商业模式的启发。

以下是该书提到的十条新经济规律及其对策：

1，去中心化、万物互联。新经济的巨大利益将来自于对分权式和自治式网络的开发和利用。在技术的网络中，这种底层集群的力量靠的是标准与规范。

策略：

  1. 最好的长期策略就是去开发那些最终会与我们的生活融为一体的产品和服务。
  2. 去发现那些曾经以为太小而被我们忽略的事物，让它们以最合适的方式去拥抱集群，形成网络。

2，网络的首要法则：价值量随着其成员关系的激增而呈现指数级增加。

策略：

  1. 通过开发网络效应而获得进步的产品、公司或技术，就会成为指数性增长的候选者。
  2. 协调次级网络，提升网络价值最快的方法就是将其余一些小网络引入到自己的网络中。
  3. 创建反馈循环。
  4. 少数有用的想法为失败的想法买单。

3，普及产生价值，每当一个封闭系统开放，它就开始与其他系统交流，于是也得到了这些系统所带来的价值。一项技术拥有越多的互联性，就能得到越多的使用和滥用。

策略：

  1. 尽可能多地接触网络，最大化你的产品和服务，比如为不同设备制造标准零件。
  2. 最大限度地给别人机会，让他人在你成功的基础上取得成功。
  3. 不要囤积产品，让它们流动起来，不要有物以稀为贵的想法。

4，免费之道：所有能被复制的东西都遵循质量提高而价格下降的曲线。科技创造需求，然后提供供给。在富足的世界里，唯一稀有的资源就是人类的注意力。

策略：

  1. 投资开发第一件商品，把免费当作定价的终极策略。
  2. 你能免费赠送什么？赠送如何进行良性循环。
  3. 次级市场，核心产品免费的同时，高价出售其他相关产品或服务。
  4. 寻找免费赠送价值的公司，追随他们。

5，网络经济中每次交易所花时间越来越少，但建立这些交易间的标准越来越难。随着非实体经济的崛起，经济领域将建立更多的标准。最终技术标准变得和法律同等重要。网络在不可逆转的颠覆世界。

策略：

  1. 网络价值最大化，让尽量多的人参与进来。
  2. 选择具有最广泛客户基础的最高质标准。
  3. 在新领域应用根深蹄固的既有标准。
  4. 无论你什么时候做科技决策，你选择连接更多的连接、更开放的系统、应用更广的标准，那么你总是正确的。
  5. 重视推广传播，发现共同利益者，把他们聚集起来，帮助扩张网络。

6，新经济中，困在局部顶峰不可避免。为了从所在山的高峰走向另一座更高的山峰，企业必须先下山。一个公司需要在自己领域外广泛的探索，有时可能要看到完全相反的领域。

策略：

  1. 借助网络，结伴而行。
  2. 创造性破坏，大胆建造。
  3. 质疑成功，站在很远的局外来思考，远到不在同一座山上。
  4. 把跨界当作一种生活方式。

7，人们占据物理地点，但经济将占据越来越多的空间。地域的影响力在减弱，而空间的影响力在增加。网络技术不但不会遏制中介，反而会促进他们的生成。网络正是中介的摇篮。网络中所有节点都是中介。网络技术不但推动了自由职业者的兴起，同时也助力了超大型组织的形成。要想建立一个购物中心，你首先需要一个社区。社区比商业重要。网络经济调动了“爱好者部落”与“信息达人”的力量。新的混乱媒体中，谣言、阴谋、妄言满天飞，这些是社区的缺陷。传统广播能够引起公众注意，网络能够发现社区，两者相辅相成。

策略：

  1. 将工作外包给网络中的其他公司。
  2. 准备好迎接潜在的网络访问高峰。

8，终身雇佣制，或是不断变更工作的现象将不复存在。新经济的主要目标是一家公司接一家公司，一个产业接一个产业地摧毁工业经济中的一切。最优系统的特点是：规矩少，临近混乱。激进式创新总出现在一团乱麻的变革途中。

策略：

  1. 勇于面对混乱边缘。
  2. 与其禁止错误的产生，不如在流变中不断的学习。
  3. 网络需要生长，而非植入。先利用小型网络进行试点，然后增加更多复杂的节点和层次。
  4. 保留核心价值，在其他方面不断激发创新与发展。

9，在网络经济中，经济规则的核心是增进联系。生产与消费合成一个动词：产销（Prosuming）。在客户与企业建立的全新关系中，因为客户参与了部分产品的制造，他们就更容易从最终的产品获得满足。谁拥有最聪明的客户，谁就能获胜。网络经济的首要任务是修复信息的对称性。

策略：

  1. 让客户变得和你一样聪明。
  2. 建立客户与客户之间的联系。
  3. 选择能够建立关系的技术，避免任何孤岛式技术。
  4. 把你的客户想象成员工。

10，与其解决问题，不如寻求机遇。每一个新发明都意味着更多新发明，而从那些新发明中又会创造出更多新的可能性。在未来，把下一件事情做对，比把同一件事情做得更好，要有意义的多。

策略：

  1. 让机器去做。
  2. 发掘出人意料的惊喜。
  3. 实现机会暴涨的最大化。

心理知识：不要公开你的个人目标！

这是企业家Derek Sivers十年前的文章和TED演讲，其观点是：

宣布你的计划会降低你完成计划的积极性。

原因是：

他人的鼓励或祝贺等社交反馈，会让大脑感觉已经接近甚至达成目标。

这似乎有违《6月读书：如何提升专注力》中的那些方法：

获得外部的赞美...向众人宣布一定要做成某件事情。

《碎片周刊第18期：如何将创意变为现实》也说：

公开想法，能促使自己将创意落地。

真的有矛盾吗？我认为，单纯争论要不要公开谈论目标这件事情没有必要，这不应该是争论的重点。重点是你要清楚目标对自己的意义，以及与他人谈论目标的意图和谈论的方式。

如果你仅仅是为了获得赞同，那还是算了，我想这才是Derek Sivers的观点。

如果是跟前辈请教实现目标的路径，这没什么不好。如果在实现过程中为了提高积极性，根据阶段性成果获取正面反馈，也是有利的。还有，如果是想利用自己的羞耻心，也并非不可。

心理知识：足够好的童年与性格陷阱

如果你觉得自己总是在重复同一类错误，这篇文章值得一读。文章指出了“强迫性重复”现象：

一百多年前，弗洛伊德在对孩子的观察中发现，孩子在经历了一件痛苦的事件之后，会在以后不自觉地反复制造同样的机会，以便体验同样的情感。

及其心理原因：

性格陷阱，又叫图式，是指我们在童年时期，因为经历一些创伤性事件，形成了对自我、对世界根深蒂固的信念。

根据英国儿童心理学家温尼科特的理念，足够好的童年包含以下内容：

  1. 基本安全感（反面：遗弃、不信任、虐待）​
  2. 与人联结（反面：情感剥夺、社交孤立）
  3. 自主性（反面：依赖、脆弱）
  4. 自尊（反面：不接纳、失败）
  5. 自我表达（反面：屈从、苛刻要求）
  6. 现实边界（反面：权利错觉）

性格陷阱的解决之道：

  1. 识别并标记自己的性格陷阱。
  2. 了解性格陷阱的童年起源，感受你内心受伤的孩子。如不堪重负，需找人帮忙。
  3. 收集反对性格陷阱的证据，在理性层面证明它无效。
  4. 给造成你性格陷阱的父母、兄弟姐妹或同伴写信。不一定要寄出，表达感受的过程才是最重要的。
  5. 认真详细地检查性格陷阱的运行模式。
  6. 打破这些模式，不断尝试。
  7. 原谅父母和伤害你的人。

文章下的评论让人心酸，有些伤害可能一辈子也无法修复。

心理知识：完美童年的误区

本文的观点似乎与上文相反，心理医生发现完美童年对成年人心理有不利影响。但稍加思考就会明白，本文所谓的完美，伴随着孩子自主能力的缺失。加州大学洛杉矶分校的精神病医生保罗·波恩发现：

很多父母会尽一切可能，避免孩子体验到哪怕一丁点的不适、焦虑或者失望...当孩子长大，面对正常的挫折，就以为事情严重出错了。

洛杉矶家庭心理师杰夫·布卢姆说：

我们把自己的需要和孩子的需要混淆了，并认为这是最佳育儿之道...一个孩子应该体验正常的焦虑，才会有适应性。

斯沃斯莫尔学院社会学教授巴里·施瓦兹：

幸福作为生活的副产品，是很棒的一个东西，但把幸福作为目标来追求，只会导致灾难。

商业知识：冠军运动员与优秀企业家的共同之处

哈佛商业评论的研究显示冠军运动员在商业领域取得成功的能力有相同之处：

  1. 看见信念与愿景的能力，将目标可视化。
  2. 抗压能力，与压力同行。
  3. 学习能力，在了解自己的技能、需求、情绪与动机的基础上创新。
  4. 信任与协作的能力，信任队友即信任自己。

写作知识：作家们为什么写作？

文中引用了10位作家，面对这个问题，有的人心直口快，有的人觉得很难回答，但总的来说，都有自己的观点。

  1. 对生活的爱恋。
  2. 渴望著作流传于世。
  3. 改变同时代人们的思想。
  4. 有东西需要表达，题目自己找上门来。
  5. 寻求生活的秩序，使生活更美好。
  6. 创作与出售。
  7. 不知如何讲述自己的故事，所以写。
  8. 因为不会做其他事情。
  9. 我写作则所以我存在。
  10. 写一般人不敢写，反一般人常态，得到幸福与快乐。

笔者认为，在心理层面，表达即意味着改变。

个人成长：7个成功职场女性犯过的错误

7个女性创业者或高级职位讲述了自己的职业误区与突破。

  1. 太在意职业安全，忽略了热情。建议：追求勇敢而不是完美，其实犯错并不会死人。
  2. 违背自己本性，例如领导风格与外表。建议：做自己。
  3. 不寻求帮助。建议：尽可能寻求帮助，找一名教练。
  4. 没有看到工作之外的世界和能力。建议：利用兴趣爱好，与组织之外的人建立联系。
  5. 在会议上说个不停。建议：从容应对，尊重他人，忠于自己。
  6. 对梦想的事业没有签订合同。建议：相信自己的直觉与力量。
  7. 两次都没有应聘上那个好工作。建议：当你为某件大事努力而没有成功，你还是会变得更强大、聪明、坚强。

大脑知识：语言是思维的脚手架

作者安娜·伊万诺娃是麻省理工学院的博士生，通过对聋哑儿童和中风失语者的案例研究得出结论：

  1. 语言使我们能够接收大量的信息，没有语言，婴儿心智不会正常发展，例如数数。
  2. 缺少语言使用环境的儿童难以想象他人的心理状态。
  3. 成年后失去语言能力并不妨碍使用数学，以及推测他人的意图，甚至创作。
  4. 神经影像学证据也支持成人的语言与认知功能分离的。

结论：

语言只是我们建造思想的工具，在建造过程中必不可少，但完成之后不是必须的。

大脑知识：意识体验不依赖于语言

意识的起源是科学未解之谜，文章说：

  1. 佛教认为所有的动物都有意识，成人和胎儿、猴子、狗、鱼，甚至蟑螂和蚊子。动物都能受苦，动物的生命都是宝贵的。
  2. 作者作为神经科学家，只能假设某些动物可能和人类一样有意识体验。而大量科学家认为人类在意识方面是独特的，人驾于所有其他生物之上。

作者的理由有三：

  1. 首先，从进化的角度来说，所有的哺乳动物都是密切相关的，胎盘类哺乳动物的共同祖先是夜间活动的毛茸茸的小动物。
  2. 其次，我们看到老鼠、狗、猴子和人的大脑有虽然有数量上的差异，但没有质量上的差异。负责疼痛的受体和途径在不同物种之间是相似的。而且，虽然人类的大脑很大，某些生物如大象、海豚和鲸鱼，有更大的大脑。其中有些动物不仅有更大的新皮质，皮质神经元的数量也是人类的两倍。
  3. 哺乳与人类近似的行为。生理测量证实：狗和人一样，心率和血压都会升高，在疼痛时都会将应激激素释放到血液中。其他训练方式都表明动物与人的行为方式非常相似。

进一步的争论是，有人认为语言是意识必须的，但也有人反对，反对者的证据如下：

  1. 病人在短暂失语症期间能继续体验这个世界。
  2. 野生儿童有严重的语言缺陷，但它并没有被剥夺体验世界的能力，虽然通常是以一种悲剧的、对他们来说难以理解的方式。
  3. 语言对我们体验世界的方式有很大的贡献，但我们对世界的基本经验并不依赖于它。

结合上一篇对语言与认知功能的研究，我认为：语言对形成正常的意识是必须的，而不同的语言环境可以形成不同的意识，这也符合对话和词汇会影响婴儿大脑发育的结论。

《碎片周刊第18期》中推送过：大脑是一个叙事器。其核心观点：动物有感受，但是没有人类的情绪意识，正是语言造成了这种差异。

学习方法：艾宾浩斯遗忘曲线

艾宾浩斯（1850-1909），德国心理学家。 记忆在最初会快速衰减，但衰减速度会逐渐平缓。

  1. 把学习总任务分散到不同时间段完成，学习效果比集中在一个时间段要好。
  2. 多次重新记忆，会有效降低知识遗忘速度。最佳复习时间点：课后，当天晚上，一周后，一个月后。
  3. 翻转式教学法，就利用了课前、课堂与课后的三个时间段克服遗忘规律。

学习方法：翻转式教学法

源自美国，The Flipped Classroom，即将传统的“课堂教学，课后作业”翻转为“课前自学，课堂实践和讨论”：

“翻转课堂”要求学生在课前通过观看教学录像，阅读教科书，或者网上查资料，弄懂课程内容，将问题和思考带到课堂。课堂上，老师默认学生对课程内容已经有了一定的了解，便不会再事无巨细地从头讲解课程内容，而是主要解答学生问题，澄清相关概念，启发课堂讨论，应用所学内容，引导学生进行深度思考，同时做相应的应用练习或者测试来进一步强调，巩固，加深，或延伸相关知识。

这要求学生进行一种主动式学习。

个人成长：什么是乔哈里视窗？

乔哈里视窗即Johari Window，被广泛用于促进自我意识和个人发展，以及更好地理解群体动力学、团队发展和人际关系，由美国心理学家Joseph luft和Harrington ingham在20世纪50年代提出。

Johari Window可以让你洞察你是如何看待自己的，别人是如何看待你的，最重要的是，两者之间的差距。

Johari Window分为4个象限：

  1. Open / 公开：你和其他人都知道的。
  2. Blind / 盲区：别人知道，你却不知道（你的盲点）。
  3. Hidden / 隐藏：你知道，但别人不知道的。它包括你有意或无意地隐藏的东西。
  4. Unknown / 未知：你和其他人都不知道的，有时是由于缺乏认知意识。

使用方法：

  1. 找到盲区，这里可能存在一些你不知道的机会。找到三个在专业上认识你至少6个月的值得信赖的人交流，看看对方对自己的看法。
  2. 分享隐藏区，这里有你的天赋、技能和兴趣。整理你的个人网站，作品与简历清单，注意你所表达的与你实际所做的有出入的地方。
  3. 针对你的表现和别人的看法之间的差距，找出你想弥补的地方，以及影响你职业表现的信息。列出签单，制定改变计划。
  4. 采取行动。

个人成长：充分利用批评和反馈的5个技巧

  1. 学会发现好的批评。真诚的反馈从你的利益出发，忽略无聊的、恶意的评判。
  2. 消除与同行的竞争或妒忌感觉。欣赏他人成就，不要把其他人的成功当作对自己的批判。
  3. 寻找可信的反馈。不一定是导师人物，你身边令你倾佩和信任并了解你的人，在失意时听听他们善意的批评。
  4. 面对最艰难的问题。有时候做不好是因为你的兴趣、专长不在那里，很可能需要转换职业。
  5. 做个有心人。利用Johari Window，找到有利于表达和改进自己的地方，有意识地弥补自己与他人看法之间的鸿沟。

个人成长：看待批判和反馈的不同方式

  1. 把反馈看作是关于提供者的信息，而不是关于你自己的信息。当你以这种方式看待反馈时，它就变成了战略上有用的信息，可以告诉我们如何吸引我们想要吸引的人：老板、客户、同事或我们作品的受众。
  2. 认识到所有与众不同的作品都会招致批评。去网络书店看看，5星级图书，依然有差评。
  3. 找到匹配的，我们最害怕的和我们认为最伤人的批评是与我们对自己的信念相匹配的批评。注意哪些过去的批评最刺痛你，也许是“傲慢”、“好但不完美”或“懒惰”，这反映了你对自己的看法...接下来，你只需要把注意力从外在的批评上转移开，对自己做出更准确、更有同情心的评价。当你改变了对自己的信念，你就不会再那么害怕或受到批评的伤害。
  4. 把事实和解释分开。通常，让我们陷入困境的不是反馈本身，而是我们对它的解释方式。比如说有几家出版商拒绝了你的图书，这是事实。但你说“我的想法没有市场价值”或者“我是个糟糕的作家”，也许是事实，也许不是。

大脑知识：成功人士的大脑

本文根据哈佛大学心理学家杰夫·布朗和神经科学家马克·芬斯克对成功人士的研究，介绍了《脑力升级手册》书中的四大成功因素对应的大脑能力：

  1. 动机，目标导向、内驱力、行动力：规划—启动—抵达。
  2. 专注力，灵活地根据不同情况调整专注水平。
  3. 逆商，复原力。
  4. 平衡情绪的能力，对自己情绪超级敏感。

组织管理：工作中的独处与社交时间

独处时间的必要性

  1. 集中精力，思考核心任务和目标，
  2. 重新充电，带领团队需要大量精力。
  3. 建立观点，思考使命、愿景和人际关系。

进行建设性休息：

建设性休息是一种健康的休息，可以补充你的能量。在工作时间，建设性的休息可以是散步，在安静的房间或在你的车里冥想，或去喝咖啡休息。在工作之余，许多人会参与看电视、和朋友喝酒、或是紧张的运动课程等活动，强迫自己休息一下。但这些做法很少能以一种可持续和令人满意的方式恢复你的身心。在大自然中散步、看书、睡觉、练习瑜伽和自由写作等活动都是真正建设性休息的例子。

社交时间的必要性

  1. 没有团队协作，就不可能是高效的团队。
  2. 与他人共事培养了与目标和集体使命的联系。
  3. 会议还可以提高参与度和积极性。

无论是一对一还是一个小组，会议都是表彰人们的辛勤工作和成就的好地方，努力做到这一点会让你的团队成员感到受到重视。

工作场所的社交活动：

社交对进步也是必不可少。不仅仅是喝咖啡聊天，团队午餐，或者团队活动，也指小组会议，一对一的谈话，以及视频通话。

找到平衡点

找到你需要多一点独处还是社交，选择一个活动，添加到每周例行公事。反复调整，经过一段时间，你就会找到平衡。
